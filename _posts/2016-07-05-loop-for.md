```python
def cuenta_vocales(s):
    '''(str) -> int
    Devuelve el nÃºmero de vocales que tiene s.
    >>> cuenta_vocales ("videotutoriales.com")
    9
    >>> cuenta_vocales("MVP")
    0'''   
    num_vocales = 0
    for caracter in s:
        if caracter in "aeiouAEIOU":
            num_vocales = num_vocales +1
    return num_vocales
```

Quan ho apliquem:

```python
>>> cuenta_vocales("videotutoriales")
8
```
