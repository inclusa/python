Podem guardar la majoria de programes Python en arxius d'extensió `.py`.

Els programas python solen dividir-se en **mòduls**. Els **mòduls** *agrupen elements relacionat entre sí*.

Els **mòduls** permeten una difisió lògica del codi, així, es poden incloure, per exemple, totes les funcions relacionades amb el *rectangle*, com ara l'àrea, el perímetre, etc.

Els **fitxers** són el suport on es guarda el codi.

Exemple de **mòdul** guardat en l'arxiu **triangle.py**:

	>>> def area_triangle(base, altura):
	...     area = (base + altura)/2
	...     return area
	... 
	>>> def perimetre_triangle(a,b,c):
	...     perimetre = (a+b+c)
	...     return perimetre
	...
