Podem alçar un servidor així:

Python 2.7

```python
python -m SimpleHTTPServer [port]
```

Python 3

```python
python3 -m http.server [port]
```

Exemple:

```python
python3 -m http.server 80
```

Caldria accedir així:

```
127.0.0.1:80
```
